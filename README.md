# vue-scaffold

A VueJs project with:
+ Sass
+ i18n
+ Jest
+ PWA
+ Vue Router
+ Vuex
+ Eslint
+ Font Awesome 5
+ Mobile Detect
+ Heroku

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
